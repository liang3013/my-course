🔺 三个不妥协

1

回测不准 = 结论全错

公告日对齐是底线。不对齐公告日 = 偷看未来财报 = 回测是假的。

2

不对不同标的用同一因子

价值股 ≠ 周期股 ≠ 成长股。周期股基本面IC < 0.05，必须换量价因子。

3

Agent 读本地 > 调外部 API

差异化在分析整合 + 观点生成，不在数据抓取。每个 Agent 背后站着一位大师。
