## 模块一：概率论与数理统计——量化不确定性的艺术

**对应知识库：** 数学5篇 + 统计学3篇 · 量化落地：IC分析 + 因子显著性检验 + VaR建模

### 核心思想
将一切投资观点表达为概率分布。量化投资的全部工作就是：把"我觉得这公司好"转化为"这个因子的IC均值0.12，年化ICIR=0.68，20日Rank IC显著不为零的概率>99%"。

### 核心内容

#### 贝叶斯法则 — Agent信念更新的数学引擎
$$P(\theta|D) = \frac{P(D|\theta) \cdot P(\theta)}{P(D)}$$

| 术语 | 含义 | 投资类比 | 量化代码中的位置 |
|------|------|------|------|
| 先验 P(θ) | 看到数据前的信念 | 大师对行业ROE的经验判断 | Black-Litterman的隐含收益率 |
| 似然 P(D\|θ) | 数据提供的证据 | 最新财报/新信息 | 样本数据计算出的因子IC |
| 后验 P(θ\|D) | 更新后的信念 | 看完新数据后的综合判断 | IC加权融合后的最终信号 |

**三级量化应用：**
1. **单因子：** 先验IC=0.10（经验）→新数据似然IC=0.15→后验IC=0.12（保守融合）
2. **多Agent辩论：** 宏观Agent悲观（先验70%看空）+ 行业Agent乐观（先验80%看多）→ 基于各自历史准确率加权→后验综合概率
3. **Black-Litterman：** 市场均衡权重=先验 → 投资者观点=似然 → 后验=最优组合权重

#### 概率分布与极值 — 风险建模的语言
- **肥尾分布（t分布/帕累托）：** 金融收益不是正态分布——极端事件比正态预测的多10-100倍
- **Copula函数：** 崩盘时所有资产的相关性突然飙升→尾部依赖比线性相关重要得多
- **VaR/CVaR/极值理论：** 定义"安全边际"的数学边界

#### 统计推断 — 从样本到总体的严谨通道

| 概念 | 公式/方法 | 量化应用 | 代码模块 |
|------|------|------|------|
| 极大似然估计(MLE) | $\hat{\theta}_{MLE} = \arg\max \prod f(x_i|\theta)$ | 估计GARCH参数·因子载荷β | `notice_align.py`参数估计 |
| 置信区间 | $\hat{\mu} \pm t_{\alpha/2} \cdot \frac{s}{\sqrt{n}}$ | 因子IC是否"真>0"而非运气 | IC显著性判断 |
| 多重检验校正 | Bonferroni·FDR(BH) | 检验1000个因子时防假阳性 | 因子筛选前的第一堵墙 |
| Spearman Rank IC | $\rho = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$ | 因子预测力衡量（抗异常值） | `multi_factor_v5.py` IC分析 |

**实战验证：** bps_growth的Rank IC_20d=0.279（n=~300周）→远超95%置信区间的临界值→真信号，不是噪声。

### 金融量化场景
- **Alpha检验三层体系：** IC分析（快筛）→分组回测（单调性）→Fama-MacBeth（纯溢价）—来自 `因子显著性检验.md`
- **风险模型：** VaR/CVaR每日期望损失上限 → 仓位硬约束
- **AI Agent：** "宏观分析师Agent"输出概率而非二元判断——「滞胀概率65%（后验），置信区间55%-73%」

## 模块二：时间序列分析与动态建模——捕捉周期的脉搏 🔥 当前优先

**对应知识库：** 统计学·时间序列分析.md · 金融计量经济学.md · HMM/状态空间待落地

### 核心思想
金融数据的核心特征：昨天的收益率和今天的不独立（波动率聚集）、趋势与周期嵌套（短/中/长周期叠加）、均值回归与动量并存。普通回归假设独立观测→金融数据完全违反→需要专门的时间序列工具箱。

### 核心内容

#### 经典时序模型

| 模型 | 核心公式 | 做什么 | 量化应用 |
|------|------|------|------|
| **AR(p)** | $Y_t = c + \phi_1 Y_{t-1} + ... + \varepsilon_t$ | 用过去预测现在 | 收益自相关检验 |
| **MA(q)** | $Y_t = \mu + \varepsilon_t + \theta_1\varepsilon_{t-1} + ...$ | 用过去冲击解释现在 | 事件冲击衰减模型 |
| **ARIMA** | AR + I(差分) + MA | 非平稳→平稳→建模 | 宏观指标趋势预测 |
| **GARCH(1,1)** | $\sigma_t^2 = \omega + \alpha\varepsilon_{t-1}^2 + \beta\sigma_{t-1}^2$ | 波动率预测 | 仓位动态调整·VIX对标 |
| **EGARCH** | 非对称GARCH | 坏消息冲击>好消息 | A股跌时波动更大 |

#### 状态空间模型与HMM — 周期识别的核心武器

**为什么HMM比美林时钟更好？**
- 美林时钟：人脑观察PMI+CPI→人为主观划分四个象限
- HMM：让模型自己从PMI/信用利差/期限利差/行业库存等数据中**无监督地发现隐藏的周期状态**

```
可观测变量：PMI、信用利差、期限利差、行业库存……
     ↓（HMM推断）
隐藏状态：State 1（复苏）· State 2（过热）· State 3（滞胀）· State 4（衰退）
     ↓（基于状态概率）
最优配置：行业权重映射 · 风格因子选择 · 仓位调整
```

**当前状态：** 37行业模板就位·造船深研完成·数学框架已建·HMM代码待启动

#### 协整与误差修正
- **协整=长期均衡关系：** 两只股票的价格各自可能随机游走，但它们的价差是平稳的（均值回归）
- **配对交易/统计套利：** 「价差偏离均值2个标准差→做空贵的·做多便宜的→等回归」
- **误差修正模型(ECM)：** 短期偏离会被长期均衡拉回→预测回归速度

### 金融量化场景
- **经济周期状态识别：** HMM动态划分→不用人判断"现在是什么周期"→模型告诉你每个状态的概率
- **轮动策略：** 周期状态概率分布→最优行业/风格/仓位权重
- **AI Agent：** "周期分析师Agent"的数学内核——每日输出当前周期位置的后验概率 + 状态转移概率矩阵

## 模块三：随机过程与连续金融——定价与风险的精髓

**对应知识库：** 数学·随机过程与随机微积分.md + 随机分析进阶.md + 蒙特卡洛数值方法.md

### 核心思想
股价不是光滑曲线。确定性世界：$dS = \mu S \cdot dt$（银行利息），真实世界：$dS = \mu S \cdot dt + \sigma S \cdot dW$（股价）。$dW$ 无处可导→普通微积分失灵→伊藤积分补位。

### 核心内容

| 概念 | 数学工具 | 金融映射 |
|------|------|------|
| **布朗运动(Wiener)** | $W_t \sim N(0,t)$，独立增量 | 股价的随机漫步成分 |
| **几何布朗运动(GBM)** | $dS = \mu Sdt + \sigma SdW$ | Black-Scholes的底层假设 |
| **伊藤引理** | $df = (f_t + \mu S f_S + \frac{1}{2}\sigma^2 S^2 f_{SS})dt + \sigma S f_S dW$ | 期权定价的数学基石 |
| **鞅(Martingale)** | $E[X_{t+1}|\mathcal{F}_t] = X_t$ | 风险中性定价——折现后价格=鞅 |
| **Feynman-Kac** | PDE↔期望 等价 | 衍生品定价的双路径 |
| **OU过程** | $dX = \theta(\mu-X)dt + \sigma dW$ | 均值回归建模（利率·波动率·配对交易） |
| **Girsanov定理** | 测度变换 | 从真实世界概率→风险中性概率 |

### 蒙特卡洛模拟 — 从点估计到分布思维

```python
# 不只给一个数字，给整个分布
# DCF估值 → 不输出「内在价值=500元」
#          输出「内在价值的概率分布：10%分位=350, 中位=500, 90%分位=720」
# → 安全边际不是个倍数，是个概率：「跌到买入价以下的概率<5%」

for sim in range(10000):
    # 随机生成增长/折现率/终值
    path_value = simulate_dcf(growth_dist, wacc_dist, terminal_dist)
    values.append(path_value)

buy_price = np.percentile(values, 30)  # 买在30%分位→70%概率赚钱
```

### 金融量化场景
- **DCF估值压力测试：** 蒙特卡洛给出完整概率分布，不用赌单一数字
- **杠杆企业=看涨期权：** 股权=E(企业价值-债务, 0)+——破产时股权归零=期权到期作废
- **组合风险管理：** 10,000条联合价格路径→VaR/CVaR不再依赖正态假设

## 模块四：最优化理论——资源配置的科学

**对应知识库：** 数学·凸优化与组合优化.md + linear_algebra_optimization_day2.md + optimization_resource_allocation.md

### 核心思想
量化投资的每个阶段都在做优化：选股→最大化Sharpe/IC · 因子组合→最小化共线性 · 资金分配→最大化风险调整收益 · 交易执行→最小化冲击成本。**优化不是锦上添花，是量化的核心引擎。**

### 核心内容

#### 从无约束到有约束

| 层级 | 方法 | 量化场景 |
|:--:|------|------|
| **无约束** | 梯度下降 | 神经网络训练·单因子参数扫描 |
| **等式约束** | 拉格朗日乘子法 $\mathcal{L}(x,\lambda)=f(x)+\lambda g(x)$ | 满仓约束(权重和=1) |
| **不等式约束** | KKT条件 | 做空限制(权重≥0)·行业偏离上限 |
| **二次规划(QP)** | $\min \frac{1}{2}x^TQx + c^Tx$ s.t. $Ax \leq b$ | **Markowitz均值方差**核心求解器 |

#### Markowitz 均值-方差优化（量化组合构建的起点）

$$\min_w \frac{1}{2}w^T\Sigma w - \lambda w^T\mu$$
$$\text{s.t. } \sum w_i = 1, \quad w_i \geq 0$$

**致命缺陷（为什么直接用会翻车）：**
1. **输入敏感：** μ的微小估计误差→w的剧烈变化→「误差最大化器」而非优化器
2. **协方差矩阵病态：** n只股票→n(n+1)/2个协方差参数→n=100时需要估计5050个参数
3. **角落解：** 无约束时权重极端集中→「分散化」名存实亡

**稳健化路径（从知识库提取）：**

| 问题 | 解法 | 知识库位置 |
|------|------|------|
| 期望收益估计不准 | Black-Litterman融合先验 | 贝叶斯统计.md |
| 协方差不稳定 | 压缩估计(Ledoit-Wolf)·EWMA | 凸优化.md |
| 权重过于集中 | 熵约束·换手惩罚·个股权重上限 | 组合管理框架.md |
| 优化器过拟合 | 重抽样优化(Resampled Optimization) | 组合优化A股实战.md |
| 实际不可交易 | 整数约束(100股整数倍)·流动性约束 | 待落地 |

#### 组合优化约束清单（全部来自实战经验）

```python
# 约束=代价函数的一部分，不是事后审查
constraints = {
    'sum_weights': 1.0,           # 满仓
    'max_single': 0.10,           # 单票≤10%
    'max_industry': 0.30,         # 单行业≤30%
    'max_turnover': 0.50,         # 换手率≤50%
    'turnover_cost': 0.003,       # 换手成本=30bp
    'tracking_error': 0.05,       # 跟踪误差≤5%
    'long_only': True,            # A股不能做空
}
```

### 金融量化场景
- **组合构建：** QP求解器输入预期μ(因子IC加权) + Σ(协方差压缩估计) → 输出精确权重
- **交易执行：** Almgren-Chriss框架最小化冲击成本 → 拆分大额订单为TWAP/VWAP
- **AI Agent：** "交易执行Agent"的内核=约束优化求解器

## 模块五：计量经济学与因果推断——寻找金融世界的"第一性原理"

**对应知识库：** 统计学·金融计量经济学.md + 因果推断.md + 因子显著性检验.md

### 核心思想
找到「盈利因子与下期收益正相关」≠「高盈利导致高收益」。相关可能是：遗漏变量（低估值才是原因）、反向因果（预期收益高→增加投资→盈利高）、选择偏差（幸存者偏差）。**因果推断=区分真因子和统计幻觉的方法论。**

### 核心内容

#### 面板数据分析 — Fama-MacBeth: 因子检验的黄金标准

**FM两步法（1973·因子投资奠基之作）：**
```
第一步（时间序列）：每只股票对因子做时间序列回归 → 估计因子暴露β
第二步（横截面）：  每个时间点对所有股票的收益 vs 因子暴露做横截面回归
                 → 对系数序列取均值+Newey-West标准误 → 因子溢价显著性
```

**为什么FM优于简单IC：**
- FM控制其他因子→测的是"纯溢价"（边际贡献）
- IC只看单因子与收益的相关性→可能被遗漏变量污染
- 学术界的因子检验标准就是FM

#### 因果推断工具箱

| 方法 | 核心思想 | 量化应用 | 何时用 |
|------|------|------|------|
| **双重差分(DID)** | 处理组变化-对照组变化=因果效应 | 政策/事件对股价的净影响 | 有处理组和对照组 |
| **断点回归(RDD)** | 刚好越过阈值=近似随机实验 | 指数纳入效应·评级变动效应 | 有明确断点规则 |
| **工具变量(IV)** | 找只影响因不影响果的工具 | 用天气/地理做盈利的工具变量 | 内生性严重时 |
| **事件研究** | 事件窗口累计异常收益CAR | 并购·分红·业绩预告的影响 | 有明确事件日 |
| **Granger因果** | 过去X有助于预测未来Y | 宏观指标是否Granger-cause行业收益 | 时序预测验证 |

**当前实战：** 3标的IC分析做的是第一层（Spearman Rank IC）和第二层（分组回测单调性）。第三层Fama-MacBeth（控制其他因子后的纯溢价）和因果推断（区分真实因子和统计幻觉）是下一步关键升级。

### 金融量化场景
- **因子有效性验证：** 不是简单看IC>0.05就算有效——必过FM的统计显著性+经济显著性双重检验
- **事件驱动策略：** DID/RDD精确评估并购、分拆、指数调整对股东价值的净影响
- **AI Agent：** "基本面分析师Agent"具备因果推理——不仅看"盈利高→收益高"，还要看"哪个因子是真正的原因"

## 模块六：贝叶斯统计与决策理论——AI Agent的大脑操作系统

**对应知识库：** 统计学·贝叶斯统计.md · 数学·probability_investment_integration.md

### 核心思想
贝叶斯框架是Agent认知架构最自然的数学表达：先验信念→观察新数据→似然评估→后验更新→期望效用最大化→行动。**整个循环就是Agent的"思考→学习→决策"过程。**

### 核心内容

#### 贝叶斯 vs 频率派 — Agent该信哪个？

| | 频率派 | 贝叶斯派 | Agent 选择 |
|------|------|------|:--:|
| 参数本质 | 固定未知的常数 | 随机变量(有分布) | ✅ 贝叶斯 |
| 不确定性 | 仅来自抽样 | 参数本身也有不确定性 | ✅ 更诚实 |
| 先验信息 | 不能用 | 可以纳入 | ✅ 大师智慧=先验 |
| 小样本 | 不可靠 | 有先验兜底 | ✅ 新上市/稀有事件 |
| 更新方式 | 重新估计 | 贝叶斯递推更新 | ✅ 持续学习 |

#### 层次贝叶斯模型 — 借用统计强度

```
参数在不同层级借用统计强度：

全局层：所有股票的"平均因子溢价"
   ↓ 先验
行业层：消费行业 vs 周期行业的因子溢价差异
   ↓ 先验
个股层：具体某只股票的因子暴露
   ↓ 后验
```

**核心价值：** 小样本（刚上市的股票/交易量小的资产）→可以从行业和全局"借"统计强度→估计不再飘忽不定

#### 计算工具

| 方法 | 原理 | Python库 | 适用场景 |
|------|------|------|------|
| **MCMC** | 马尔可夫链采样从后验分布中抽样 | PyMC/Pyro | 中等复杂度模型 |
| **变分推断(VI)** | 用简单分布近似复杂后验 | Pyro/TFP | 大规模/实时场景 |
| **粒子滤波** | 用带权粒子近似状态分布 | 自实现/filterpy | 在线状态追踪（HMM升级版） |

#### Agent认知循环 — 贝叶斯全链路

```
1. 先验信念
   宏观Agent: 「当前处于复苏期(先验概率60%)」
   ↓
2. 观察新数据
   新PMI=50.8·信用利差收窄·行业库存下降
   ↓
3. 似然评估
   P(这些数据 | 复苏) vs P(这些数据 | 衰退)
   ↓
4. 后验更新
   复苏概率 60%→73%  ←  信念更新，不是推翻
   ↓
5. 期望效用最大化
   复苏环境最优仓位=股票75%+债券15%+现金10%
   ↓
6. 执行行动
   调仓指令生成
```

### 金融量化场景
- **多源信息融合：** 宏观+行业+个股三层贝叶斯网络→每个层级输出概率分布→最终融合
- **动态参数更新：** Agent根据新市场数据每天更新模型参数——「每天变得更聪明一点」
- **Black-Litterman：** 均衡收益=先验 + 投资者观点=似然 → 后验=最优权重 — 贝叶斯在组合管理中最高光的应用

## 模块七：机器学习数学基础——从XGBoost到深度学习

**对应知识库：** 统计学·机器学习基础.md + 统计学习理论.md · 数学·算法与数据结构.md

### 核心思想
量化投资是ML最优落地场景之一：结构化数据（价量+财务+宏观）、明确目标（未来收益）、海量历史（回测天然验证）、噪声充沛（考验过拟合防御）。

### 核心内容

#### 偏差-方差分解：ML在金融中的核心矛盾

$$\text{MSE} = \underbrace{\text{Bias}^2}_{\text{欠拟合}} + \underbrace{\text{Variance}}_{\text{过拟合}} + \underbrace{\sigma^2_{\varepsilon}}_{\text{不可约误差}}$$

**金融ML的独特挑战：信噪比极低**（金融数据的不可约误差远大于图像/文本）→过拟合是默认状态，不是意外。

#### 过拟合防御体系（量化必备）

| 层级 | 防御 | 具体做法 | 知识库位置 |
|:--:|------|------|------|
| **数据** | 时间序列交叉验证 | 不用随机K-Fold·用滚动窗口·防前视泄漏 | 统计学习理论.md |
| **模型** | 正则化(L1/L2) | 惩罚复杂模型·自动特征选择 | 机器学习基础.md |
| **特征** | 维度灾难防御 | 每增加一个因子需额外~50个观测 | — |
| **验证** | 样本外测试 | 最后20%数据从未参与任何训练 | 因子显著性检验.md |
| **生产** | 衰减监控 | IC一旦跌破阈值→自动触发重训练 | 量化投研实战工作流.md |

#### 梯度提升树（XGBoost/LightGBM）— 量化主力

- **优势：** 天然处理缺失值·特征重要性输出·非线性关系·过拟合控制强·无需特征标准化
- **在量化中的正确打开方式：** 不是直接预测收益率→而是合成非线性因子→再做线性IC分析→线性组合→保留可解释性
- **当前状态：** 20本原著中《机器学习与资产定价》覆盖此块·D2待下载

### 金融量化场景
- **非线性因子挖掘：** XGBoost从原始数据中自动发现非线性关系→输出因子→IC验证
- **过拟合检测：** 回测"太好看"→极大概率过拟合→交叉验证+样本外确认
- **AI Agent：** "ML分析师Agent"的职责不是替代线性模型→是发现线性模型看不到的非线性模式
