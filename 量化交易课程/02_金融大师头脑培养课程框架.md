>
> 📚 知识底盘：21位大师 · 20个知识库 · 覆盖全球→本土、微观→宏观、理论→实战

## 模块一：商业语言与会计艺术——像巴菲特一样读财报

**对应大师：** 巴菲特（226文件·6.7MB·70年股东信）+ 唐朝 + 段永平

### 核心思想
会计是商业的语言，更是发现真相的侦探工具。巴菲特从1956年第一封合伙人信写到2025年——70年的投资通信是理解"如何读财报"的最佳教材。

### 知识库资源

| 资源 | 规模 | 核心内容 |
|------|------|------|
| 巴菲特股东信 | 98篇·4.1MB | 1956-2025，从"捡烟蒂"到"以合理价格买伟大公司"的完整进化 |
| 巴菲特公司档案 | 61篇·1.0MB | 61家主要投资标的/收购/持股公司的深度分析 |
| 巴菲特投资概念 | 49篇·1.4MB | 护城河·安全边际·能力圈·管理层·内在价值 全体系 |
| 唐朝知识库 | — | 财报实务+估值简化「老唐估值法」（三年后合理估值=当年净利润×1/无风险利率） |
| 段永平语录 | 归纳版 | 「买股票就是买公司」+ 本分经营哲学 + 中国价值投资实操 |

### 会计分析的精髓
- **权责发生制的陷阱与调整：** 收入确认激进（提前确认）、费用资本化（推迟确认）、商誉减值时机——巴菲特70年信中反复出现的主题
- **盈利质量的量化维度：** 经营现金流/净利润长期匹配度（应计比率）、非经常性损益占比——段永平「现金流好的公司才是真的好」
- **管理层行为的财务印记：** 库存周转率突变、应收账款激增、折旧政策变更→反向推断管理层动机与诚信度
- **ROE杜邦分解：** 净利率×周转率×杠杆——巴菲特最爱的指标，三因子拆解看清盈利驱动力

### 估值建模 — 量化接口
- 从零为任何商业模式搭建动态DCF（FCFF/FCFE二选一 + WACC + 三情景终值）
- AI Agent自动生成数千种假设组合，寻找安全边际的"下限"而非"最优点"
- 唐朝简化版：「三年后合理估值 = 当年净利润 × 1/无风险利率」 → 打七折 = 买点

### 核心书目（知识库已收录）
《巴菲特致股东的信》·《证券分析》（格雷厄姆）·《巴芒演义》·《价值投资实战手册》（唐朝）·《段永平投资语录归纳版》

## 模块二：周期、宏观与市场心理学——马克斯与索罗斯之间架桥

**对应大师：** 霍华德·马克斯 + 索罗斯（9文件·56KB·5/20新建）+ 达利欧 + 周金涛

### 核心思想
理解周期的位置，洞悉市场自身的偏见如何改变周期。"我们可能永远不知道要去哪里，但最好知道我们现在在哪里"（马克斯）。而索罗斯告诉你：市场不仅反映现实，还主动塑造现实——投资者的偏见可以改变基本面。

### 多层次周期框架（马克斯视角）
- **五大支柱：** 第二层次思维 · 周期理论 · 风险哲学 · 钟摆理论 · 逆向投资
- **周期识别：** 企业盈利周期（微观）、信贷周期（最重要——「信贷窗口打开的宽度比价格更重要」）、风险态度周期（贪婪↔恐惧钟摆）
- **第二层思维：** 不只思考"公司好不好"，而是想"有多少人认为它好，这个共识在价格中反映了多少"——这正是量化IC分析背后的哲学

### 反身性与繁荣-萧条（索罗斯视角，知识库5/20新建）
- **反身性理论核心：** 认知功能（理解世界）+ 参与功能（改变世界）→ 双向反馈环路 → 自我强化→最终自我毁灭
- **繁荣-萧条八阶段：** 初始趋势→趋势被认可→进入测试期→信念增强→现实与预期背离→转折点→自我强化下行→崩溃
- **易错性原理：** 金融市场永远是有偏见的——关键不是"偏见是否存在"，而是"偏见何时反转"
- **量化接口：** 繁荣-萧条八阶段 → 可量化为动量加速/减速信号 + 成交量确认 + 估值偏离度

### 达利欧的宏观机器
- **债务周期框架：** 短期债务周期（5-8年）+ 长期债务周期（50-75年）→ 全天候策略的底层逻辑
- **三大宏观驱动力：** 生产率增长 + 短期债务周期 + 长期债务周期
- **原则驱动：** 「痛苦+反思=进步」→ 量化系统中的回测=痛苦，因子重校=反思

### 核心书目
《投资最重要的事》《周期》（马克斯）·《金融炼金术》《超越金融》（索罗斯）·《原则》《债务危机》（达利欧）·《涛动周期论》（周金涛）

## 模块三：人类误判心理学与行为金融——芒格的核心武器

**对应大师：** 芒格 + 豪泽尔 + 卡尼曼（间接·通过芒格）

### 核心思想
所有投资错误都源于认知偏差。"在手裡只有錘子的人看來，所有問題都是釘子"（芒格）。建立检查清单避免认知偏差，已超越90%的投资者。而豪泽尔告诉你：投资成功更多取决于"你怎么管自己"而非"你怎么分析公司"。

### 芒格的25个心理倾向（精选10大杀伤力最大的）

| # | 倾向 | 投资表现 | 量化防御 |
|:--:|------|------|------|
| 1 | **激励超级反应** | "永远不要问理发师你是否需要理发"——分析师推荐自家持仓 | 剔除分析师覆盖密度异常高的信号 |
| 2 | **社会认同/从众** | 周期顶部追涨、底部割肉——最大亏损来源 | 信号密度>40%时自动降权 |
| 3 | **厌恶剥夺/损失厌恶** | 持仓让我们变傻，亏损不舍得割 | 止损规则程序化·禁止人工干预 |
| 4 | **确认偏差** | 只看支持自己结论的证据 | Agent辩论：强制多空双方便签 |
| 5 | **过度自信** | 回测结果"太好看"时的盲目信任 | 前视偏差检查+样本外验证为底线 |
| 6 | **锚定效应** | 被买入价锚定，无法客观评估当前价值 | 定期重置成本基准·用分位替代绝对值 |
| 7 | **承诺升级** | 越跌越买→不是价值投资，是沉没成本谬误 | 加仓需独立触发条件·不因"已经亏了"而加 |
| 8 | **易得性偏差** | 最近发生的事被过度放大 | 用10年滚动窗口而非最近1年做因子IC |
| 9 | **叙事谬误** | 为随机波动编造"故事" | 要求每个信号有统计显著性·拒绝"听起来合理" |
| 10 | **Lollapalooza效应** | 多个偏差同时共振→毁灭性后果 | 极端行情多偏差联动检测→自动降仓 |

### 豪泽尔的金钱心理学
- **没人对"理性"有统一标准**——每个人的风险偏好由其人生经历塑造
- **复利最大的敌人是"打断"**——不是回报率，是持有时间
- **知足是超能力**——"够了"是投资中最被低估的词

### 量化应用
- Agent奖惩函数中模拟非对称风险厌恶（前景理论）
- Agent辩论时设计"故意错误锚定"测试 → 训练抗干扰能力
- "豪泽尔Agent"专门负责评估：当前决策是理性还是情绪驱动的

### 核心书目
《穷查理宝典》（芒格）·《思考，快与慢》（卡尼曼）·《金钱心理学》（豪泽尔）·《影响力》（西奥迪尼）

## 模块四：跨学科思维模型栅格——芒格的终极智慧

### 核心思想
从多个学科提取约100种最强大的思维模型，组成决策网格。芒格50年搭档改变巴菲特的投资风格——从格雷厄姆的"捡烟蒂"升级为"以合理价格买伟大公司"。

|        | 贝叶斯更新  | 先验(大师观点) + 似然(新数据) → 后验(更新信念) | IC衰减→因子降权        |
|        | 凯利公式   | 最优仓位 = (胜率×盈亏比-败率)/盈亏比        | 仓位上限硬约束          |
| **物理** | 临界点    | 因子拥挤度阈值→风格崩溃预警                | 水在100°C之前都是水     |
|        | 熵增     | 策略α必然衰减→需持续迭代                 | 不维护的系统必然退化       |

- 跨学科检查清单：每个投资决策必经数学·物理·生物·经济·心理·哲学六学科交叉核验

### 核心书目
《穷查理宝典》《探索智慧：从达尔文到芒格》《模型思维》（斯科特·佩奇）· 芒格知识库107文件全量

## 模块五：逆向、秃鹫与特殊情况投资——非对称收益的艺术

**对应大师：** 老木匠（A股便宜+逆向+周期股）+ 塔勒布（反脆弱+杠铃策略）+ 卡拉曼/格林布拉特（安全边际·特殊情况）

### 核心思想
最好机会出现在市场被恐惧笼罩、价值极度扭曲或存在复杂博弈的领域。塔勒布说：「不要试图预测黑天鹅，要让自己在黑天鹅中受益」。

### 塔勒布的反脆弱体系
- **黑天鹅三特征：** 稀有性 + 极端冲击 + 事后可解释性（但事前不可预测）
- **反脆弱：** 不是"抗打击"（坚韧），而是"越打击越强"——波动和混乱反而让你获益
- **杠铃策略：** 90%极度安全（现金/国债）+ 10%极度冒险（期权/早期投资）→ 下行有限，上行无限
- **实践高于理论：** 塔勒布蔑视"只说不做"的学者——投资理念必须在真实市场中检验

### 老木匠的A股实战智慧
- **核心：** 便宜 + 逆向 + 周期股——在A股用"便宜"赚钱的极致实践者
- **忍耐与时机：** 不是"什么时候买"，是"什么时候值得等"——A股最被低估的品质

### 量化接口
- 秃鹫投资→破产重组价值博弈模型
- 事件驱动→并购套利的概率+时间加权期望收益
- 杠铃策略→组合层面的极端尾部对冲（深度虚值看跌期权）

### 核心书目
《反脆弱》《黑天鹅》《随机漫步的傻瓜》（塔勒布·知识库收录）·《安全边际》（卡拉曼）·《你也可以成为股市天才》（格林布拉特）

## 模块六：金融史、泡沫与崩溃——历史的韵律

**对应大师：** 金德尔伯格 + 明斯基 + 达利欧（债务危机史）+ 周金涛（康波周期）

### 核心思想
太阳底下没有新鲜事，人性从未改变。达利欧研究了过去500年所有大国的兴衰——每个帝国的债务周期高度相似。周金涛说：人生就是一场康波——财富的波动不是线性的，是周期性的。

### 经典泡沫解剖（从知识库提取）

| 泡沫 | 核心机制 | 现代映射 |
|------|------|------|
| 南海泡沫(1720) | 特许权炒作+全民投机 | SPAC狂潮(2020-21) |
| 1929大崩盘 | 杠杆+保证金交易+银行混业 | 2008次贷/2021杠杆牛 |
| 2008金融危机 | 复杂金融工程+评级失灵+信用崩塌 | 任何"这次不一样"的新金融产品 |
| LTCM(1998) | 诺奖得主+过度杠杆+模型盲点→40亿瞬间蒸发 | AI量化模型过度自信 |
| 日本泡沫(1989) | 资产价格永涨信仰+宽松货币 | 任何"XX永远涨"的信仰 |

### 读史目的
- 训练对"这次不一样"的本能警觉——这四个字是金融史上最贵的四个字
- 达利欧式模板：每次危机前都有相同的信号（债务/GDP飙升+资产泡沫+政治极端化）
- 明斯基时刻：稳定本身孕育着不稳定——越久的平静，越大的风暴

### 核心书目
《疯狂、惊恐与崩溃》（金德尔伯格）·《稳定不稳定的经济》（明斯基）·《债务危机》《原则》（达利欧）·《涛动周期论》（周金涛）

## 模块七：理性、心性与决策纪律——打造你自己的"原则"

**对应大师：** 达利欧（原则系统）+ 段永平（本分+不做清单）+ 德鲁克（有效性思维）+ 纳瓦尔（判断力=财富）

### 核心思想
将前面六个模块的知识转化为个人和AI系统的行动准则。达利欧说：「痛苦+反思=进步」——量化系统的回测=痛苦，因子重校=反思。段永平说：「不做清单比做什么清单更重要」——量化中踢出IC<0.05的因子就是"不做"清单。

### 十大核心原则（从21位大师提炼的公约数）

**核心设计原则：**
- 每个Agent背后有完整的知识库支撑（不是prompt调教，是知识库驱动）
- 芒格Agent的核心职责不是给答案，是"系统性批判"——找出其他Agent遗漏的盲区
- 风控Agent拥有一票否决权——不管理由多充分，超过风险预算就否决
- 董事长Agent不是"人"，是加权投票算法+贝叶斯融合——先验来自大师知识库权重，似然来自数据信号

### 个人投资检查清单（葛文德式——事前·事中·事后）

**事前（下单前必过）：**
- [ ] 护城河：这个生意的竞争优势在5年后还在吗？（巴菲特）
- [ ] 安全边际：买入价至少低于内在价值30%（格雷厄姆）
- [ ] 管理层：这群人值得信任吗？（德鲁克·段永平—本分）
- [ ] 周期：我们现在在周期的什么位置？（马克斯）
- [ ] 反身性：市场对这个标的的共识在加速吗？（索罗斯）
- [ ] 逆向：如果我是空头，我会怎么攻击这个假设？（芒格）
- [ ] 尾部风险：最坏情况是什么？能承受吗？（塔勒布）
- [ ] 我遗漏了什么学科的视角？（芒格多元思维）

**事后（每笔交易复盘·达利欧式日志）：**
- 这个决策是理性判断还是情绪驱动？（豪泽尔）
- 有哪些我当时不知道但应该知道的？
- 大师中谁会对这个决策说"不"？
