## 一、知识储备：四大支柱

构建"T"型知识结构，横跨多领域，在AI Agent与量化金融的结合点上有深度理解。

### 1. 金融与投资理论（地基）

#### 价值投资内核
- **深度基本面分析：** 会计勾稽关系、盈利质量（经营现金流/净利润比）、商誉与无形资产处理
- **估值方法论：** DCF（FCFF/FCFE + WACC + 终值敏感性）、剩余收益模型（RIM）、PE/PB适用边界与失效条件
- **护城河量化：** 品牌溢价率、转换成本量化指标、网络效应密度——将定性护城河转化为可计算特征
- **大师智慧映射：** 每个量化因子背后站着一位大师——ROE→巴菲特、市赚率→段永平/唐朝、安全边际→格雷厄姆

#### 周期轮动理论
- **多周期嵌套：** 美林时钟（增长-通胀） + 货币-信用框架 + 基钦（库存40月） + 朱格拉（资本开支7-10年）
- **行业传导链：** 上游资源→中游制造→下游消费的非对称传导时滞 → 因子在不同周期阶段的有效性切换
- **风格因子轮动：** 价值/动量/质量/低波/规模在不同宏观象限下的表现差异 → RL自动学习切换
- **行为金融周期：** 周期的极端点 = 贪婪高点（ROE顶峰=风险最高） vs 恐惧低点（ROE低谷=机会最大）

### 2. 数学与统计学（工具）

- **概率论：** 贝叶斯推断——先验（大师观点）+ 似然（数据信号）→ 后验（融合决策）
- **时间序列分析：** ARIMA/GARCH 波动率建模、协整分析（配对交易）、卡尔曼滤波（动态因子暴露）
- **计量经济学：** 面板回归（Fama-MacBeth横截面）、因果推断（DID/RDD识别策略有效性）
- **最优化理论：** 带约束凸优化（交易成本/换手率/集中度）+ 重抽样稳健优化
- **信息论：** IC（信息系统）→ ICIR（信息比率）→ 因子衰减半衰期 → 因子生命周期管理

### 3. 计算机科学与AI（核心引擎）

#### 机器学习基础
- **有监督学习：** XGBoost/LightGBM 非线性因子合成 → 样本外IC验证 → 过拟合防御（时间序列交叉验证）
- **无监督学习：** HMM/GMM 从宏观数据中无监督划分隐藏的"周期状态"——让模型自己发现轮动规律
- **NLP与知识图谱：** 研报情感 → 因子增强信号；产业链图谱 → 行业因子构建；LLM微调 → 财报解读Agent

#### AI Agent 架构
- **认知架构：** 记忆（短期/长期/工作记忆） → 规划（任务分解树） → 反思（自我批评回路） → 工具调用
- **多智能体协作（MAS）：** 财务多/空辩论Agent + 行业分析师Agent + 宏观策略师Agent → 裁判Agent融合信号
- **Agent与量化接口：** Agent读本地研报（差异化价值） > 调外部API（不稳定） → 输出结构化信号 → 回测验证
- **课程体系：** [[AI Agent工作流架构师课程/|四级课程]]（初级→专家，约67术语，24 SVG图表）— 建设Agent团队的底层知识

#### 量化工程工具链
- **回测引擎：** 向量化（快速验证） → 事件驱动（准确回测·涨跌停/停牌/T+1）
- **公告日对齐：** A股用NOTICE_DATE → 港股用规则推定（年报4/30·中报8/31·Q1 5/15·Q3 11/14）
- **信号生成：** IC加权 > 等权 · 周频 > 日频 · 动态分位阈值 > 固定阈值

### 4. 软件工程与MLOps（交付管线）

- **数据管道：** 稳健ETL（实时/批量双模）· 异构数据清洗对齐 · 港股字段映射（ROE_YEARLY→roe等）
- **回测系统：** 事件驱动型引擎 · 前视偏差防御（公告日对齐为底线）· 幸存者偏差/过拟合检查
- **MLOps闭环：** Docker包装策略 → MLflow追踪实验 → A/B测试新旧策略 → 自动重训练（检测IC衰减）
- **组合管理系统：** 每日定时触发 → 各Agent异步运行 → "晨会"信号融合 → 生成调仓清单 → API下单

## 三、项目经验：分阶段构建专家系统

### 阶段一：地基 — 把回测做对（当前阶段）

#### 项目1：智能估值与财务异常检测系统
- **目标：** AI替代传统人工财务分析
- **输入：** 21家公司全量数据（K线+财务+分红+研报）
- **技术：** NLP抽取商业模式 → 知识图谱比对同业 → XGBoost财务造假预测
- **输出：** AI自动生成个股估值压力测试报告（乐观/基准/悲观三情景）

#### 项目2：多维度经济周期状态识别与轮动策略 🔥 优先
- **目标：** 比美林时钟更精细的动态轮动——让模型自己发现周期状态
- **技术路径：**
  1. 选取PMI、信用利差、期限利差、行业库存等宏观指标
  2. HMM/动态因子模型识别4-6个隐藏周期状态
  3. 每个状态→最优行业/风格映射 → 轮动信号
  4. 37行业因子模板接入（造船深研经验泛化）
- **核心验证：** 样本外稳健性 + 经济学可解释性（每个隐藏状态必须能命名）

#### 项目3：公告日对齐 + 事件驱动回测引擎（已初步完成 ✅）
- **已完成：** A股NOTICE_DATE对齐 + 港股规则推定 → 3标的6轮迭代验证
- **待深化：** 涨跌停/停牌模拟完整性 · 多标的横截面Fama-MacBeth检验

### 阶段二：进阶 — Agent介入决策

#### 项目4：基于Agent的投研多空辩论系统
- **目标：** 模拟投研会——多个LLM Agent独立研究、相互挑战，而不是一个模型自说自话
- **架构：**
  1. **财务多Agent**：收集一切"这个公司好"的证据（ROE趋势/现金流/管理层）
  2. **财务空Agent**：收集一切"这个公司有问题"的证据（应收账款异常/关联交易/利润质量）
  3. **行业分析师Agent**：竞争格局、产业链位置、市场份额趋势
  4. **宏观策略师Agent**：当前周期是顺风还是逆风
  5. **裁判Agent**：汇总结构化论点/论据/信心评分→生成最终决策+少数派报告
- **回测验证：** Agent信号 vs 纯因子信号，对比IC和超额收益
- **Agent 设计理念：** 读本地研报 > 调外部API（外部源不稳定），差异在分析整合+观点生成

#### 项目5：自适应周期轮动的强化学习Agent
- **目标：** 让Agent自己学会风格/行业切换——不是人写规则
- **RL环境设计：**
  - 状态空间：周期象限 + 动量 + 波动率 + 行业相对强度 + 因子拥挤度
  - 动作空间：各行业/风格的配置比例
  - 奖励函数：风险调整收益（夏普比率）- 换手惩罚 - 集中度惩罚
- **关键约束：** 设计"投资常识惩罚函数"——加杠杆 ?、追涨停 ?、卖在恐慌 ? → 大幅扣分
- **对比基准：** 固定权重多因子策略、等权行业配置

### 阶段三：大成 — 端到端系统

#### 项目6：端到端自主投资决策与执行系统
- **系统集成：**
  - 项目2 周期状态 → 设定当前宏观象限
  - 项目1 个股估值 → 筛选候选池
  - 项目4 多Agent辩论 → 每只候选标的信号
  - 项目5 RL轮动 → 行业/风格权重
  - 风控Agent → 一票否决权（监控波动率/最大回撤/集中度）
- **每日流程：**
  ```
  08:00 数据采集自动触发
  09:00 各Agent异步运行（财务·行业·宏观）
  09:20 "晨会"信号融合—裁判Agent生成调仓清单
  09:25 风控Agent审查—波动率/回撤/仓位上限检查
  09:30 调仓指令自动执行
  ```
- **回测闭环：** 每季度复盘 → IC衰减检测 → 因子/Agent自动重校准

## 五、方法论积累：从实战中提炼的规则

### IC 分析铁律
| IC 绝对值 | 含义 | 行动 |
|:--:|------|------|
| > 0.15 | 强预测力 | 高权重 |
| 0.05-0.15 | 有效 | 保留·低权重 |
| < 0.05 | 噪声 | **踢出** |

### 公告日对齐铁律
- 不对齐公告日 = 回测是假的
- 对齐后必须重跑所有IC → 前视水分全部暴露
- A股用 NOTICE_DATE（数据中已有）· 港股用规则推定

### 策略参数体检表
| 指标 | 健康范围 | 预警 |
|------|:--:|------|
| 信号密度 | 10-20% | >40% 无区分度 |
| 交易次数/年 | 0.5-3次 | >5次 噪音交易 |
| 单边最长持有 | 3-18月 | <1月 过度交易 |

### 回测信任度检查清单
- [ ] 起始价 > 5元（前复权不失真）
- [ ] 对齐了公告日（未偷看未来财报）
- [ ] 模拟了涨跌停（涨停买不到/跌停卖不掉）
- [ ] 扣除了完整交易成本（佣金+印花税+冲击成本）
- [ ] 控制了因子中性化（市值/行业暴露已知）

### 因子有效性升级路径
```
日频IC薄 → 周频才有效（bps_growth 日频0.16→周频0.28）
基本面因子有效 → 价值股（腾讯·茅台·消费）
基本面因子失效 → 周期股 → 必须换量价因子（船舶·煤炭）
IC加权 > 等权 · 动态分位 > 固定阈值
```

## 总结

**螺旋上升飞轮**：学习金融理论 → 量化建模 → AI Agent自动化决策 → 回测/实盘检验 → 从失败中重构方法论

**真正的护城河**：对价值投资和周期的深度理解赋予量化策略灵魂——每个因子背后站着一位大师（ROE→巴菲特、市赚率→段永平、安全边际→格雷厄姆），每个Agent背后都应站着一位投资大师的影子。

**三个不妥协**：
1. 回测不准 = 所有结论都是错的 → 公告日对齐是底线
2. 不对不同标的用同一因子 → 价值股≠周期股≠成长股
3. Agent读本地数据 > 调外部API → 差异化在分析，不在抓取
